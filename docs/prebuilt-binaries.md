# Install pre-built binaries

The easiest way of installing the Proton command line tools is to use our set of pre-built binaries.

### Ubuntu 18.04

```bash
wget https://github.com/ProtonProtocol/proton/releases/download/v2.0.5/proton_2.0.5-1-ubuntu-18.04_amd64.deb
sudo apt install ./proton_2.0.5-1-ubuntu-18.04_amd64.deb
```

### Ubuntu 16.04

```bash
wget https://github.com/ProtonProtocol/proton/releases/download/v2.0.5/proton_2.0.5-1-ubuntu-16.04_amd64.deb
sudo apt install ./proton_2.0.5-1-ubuntu-16.04_amd64.deb
```

### RPM-based (CentOS, Amazon Linux, etc.)

```bash
wget https://github.com/ProtonProtocol/proton/releases/download/v2.0.5/proton-2.0.5-1.el7.x86_64.rpm
sudo yum install ./proton-2.0.5-1.el7.x86_64.rpm
```
